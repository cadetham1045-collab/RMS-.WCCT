# RMS - Results Management System (WAPDA CADET COLLEGE TARBELA)

Simple Flask app for classes 8-12 with admin, teacher, and student portals.

Quick start

1. Create a virtualenv and install:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Run the app:

```bash
python app.py
```

3. Open http://localhost:5000
